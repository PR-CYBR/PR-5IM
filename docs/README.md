# Documentation

This directory contains documentation for the PR-5IM (Puerto Rico System Integration & Simulation Matrix) framework.

## Contents

- Framework architecture and design documentation
- API documentation
- User guides and tutorials
- Configuration guides
- Development documentation

## Getting Started

Documentation is organized by topic:
- **Architecture**: High-level system design and component interactions
- **User Guides**: How-to guides for using PR-5IM
- **API Reference**: Detailed API documentation for developers
- **Configuration**: Guide to configuration options and modes
- **Examples**: Example simulations and use cases
